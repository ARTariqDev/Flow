An all in one planner app allowing you to set timers,events,alarms or make a to-do list to manage your day to day "Flow". 

Currently I have implemented 3/5 functions, those being:
  --> Calendar (Events have yet to be added)
  --> Timers
  --> Alarms
The completed app will have:
  --> Calendar (with the ability to add Events)
  --> Timers
  --> Alarms
  --> Stopwatch
  --> To-Do List
  --> A "Dynamic" home screen that shows Any upcoming events,your current/remaining To-Do's, any upcoming Alarms and active Timers or an active stopwatch.
